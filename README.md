🛠 Especificaciones Técnicas del ADN de Sierra Leona
Este contrato documenta la interacción entre pigmentos vegetales y aceites vírgenes:

Solubilidad de la Hoja (uint256): Cuantifica la integración de las hojas (patata o mandioca) tras el procesado mecánico. En el Potato Greens, un valor de 96 indica que las paredes celulares se han roto lo suficiente para liberar clorofila y espesar la salsa sin dejar fibras gruesas.

Pureza del Aceite de Palma (uint256): Registra la calidad de la extracción. Los valores altos (9-10) representan el uso de aceite virgen, rico en carotenos, que actúa como el vehículo principal para los sabores liposolubles del pescado ahumado.

Uso de Pescado Ahumado (bool): Validador que identifica la presencia de proteínas tratadas térmicamente para aportar umami y profundidad al perfil molecular del guiso.

Sistema de Feedback: Funciones de darLike y darDislike integradas para la gobernanza culinaria on-chain.
